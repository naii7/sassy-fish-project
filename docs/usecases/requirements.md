# Functional Requirements

## 1. User Login

1. The system should allow users to log in using a username and password.  
2. After login, the system should show their profile page.

---

## 2. Profile

3. The system should display the user's profile picture.  
4. The system should display the user's description/bio.  
5. The system should display the user's favourite posts.  
6. The system should display all posts created by the user.  
7. The system should allow filtering posts by tag.
8. The system should allow the user to create a new post using a "+" button.  

---

## 3. Post Creation

9. The system should allow users to upload an image.  
10. The system should allow users to enter a title.  
11. The system should allow users to select one tag from a predefined list (ENUM).  
12. The system should allow users to select a date.  
13. The system should allow users to select a star rating.  
14. The system should allow users to mark a post as favourite.  
15. The system should allow users to write a personal comment/description.  
16. The system should save the post and display it on the user profile.

---

## 4. Tags

17. The system should provide the following predefined tag categorized by colors:
- misfortune
- life crisis
- gym
- wins
- main character moment
- toilet
- travel
- food
- film
- book
- music
- videogame
- other
18. the system should display the tags using colors to indicate their category.
19. The system should allow users to filter posts by selecting a tag.
